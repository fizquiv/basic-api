API Endpoint Reference

This API allows you to manage game items and users. Each endpoint is documented below.

Method Endpoint Description
GET /api/items Returns a list of all items in the catalog.
GET /api/items/:id Fetches a specific item by ID (e.g., sword01).
POST /api/items Creates a new item. Requires id, name, type, and effect.
PUT /api/items/:id Updates an item by ID.
DELETE /api/items/:id Deletes an item by ID.
GET /api/users Returns a list of all users, including item details.
GET /api/users/:id Fetches a user by ID and includes their items.
POST /api/users Registers a new user. Requires id, name, email, and optional item IDs.
PUT /api/users/:id Updates a user's name, email, or items.
DELETE /api/users/:id Deletes a user by ID.
